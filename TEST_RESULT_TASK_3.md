# 1) Прямой ответ
# Поиск второго по величине числа

## Массив
`[3, 1, 4, 1, 5, 9, 2, 6]`

## Решение

### Шаг 1 — Сортировка по убыванию
Отсортируем массив: `[9, 6, 5, 4, 3, 2, 1, 1]`

### Шаг 2 — Убираем дубликаты
Уникальные значения: `[9, 6, 5, 4, 3, 2, 1]`

### Шаг 3 — Берём второй элемент
- **1-е место:** `9`
- **2-е место:** `6` ✅

## Ответ

> **Второе по величине число равно `6`**

---

## Пример кода (Python)

```python
arr = [3, 1, 4, 1, 5, 9, 2, 6]
second_max = sorted(set(arr), reverse=True)[1]
print(second_max)  # 6
```

# 4) Группа экспертов

# Поиск второго по величине числа в массиве [3, 1, 4, 1, 5, 9, 2, 6]

---

## 🔍 Аналитик

**Разбор условия:**

- Дан массив: `[3, 1, 4, 1, 5, 9, 2, 6]`
- Нужно найти **второе по величине** число
- Ключевой вопрос: считаем ли мы **уникальные значения**?
    - Например, `[5, 5, 3]` — второе по величине это `3`, а не `5`
- В массиве есть **дубликат** числа `1` — это важно учесть
- **Уникальные значения:** `{1, 2, 3, 4, 5, 6, 9}`
- Отсортированный по убыванию: `9 → 6 → 5 → 4 → 3 → 2 → 1`

---

## ⚙️ Инженер

**Решение на Python:**

```python
def second_largest(arr):
    unique_sorted = sorted(set(arr), reverse=True)  # убираем дубли и сортируем
    return unique_sorted[1] if len(unique_sorted) >= 2 else None

arr = [3, 1, 4, 1, 5, 9, 2, 6]

# Шаг 1: убираем дубликаты → {1, 2, 3, 4, 5, 6, 9}
# Шаг 2: сортируем по убыванию → [9, 6, 5, 4, 3, 2, 1]
# Шаг 3: берём индекс [1] → 6

print(second_largest(arr))  # ✅ Результат: 6
```

**Эффективное решение за O(n) без сортировки:**
```python
def second_largest_fast(arr):
    first = second = float('-inf')
    for num in arr:
        if num > first:
            second = first
            first = num
        elif num > second and num != first:
            second = num
    return second
